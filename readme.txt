== Description ==

This plugin support payment gateway Momo (momo.vn) for WooCommerce.  
Auto confirm and redirect after user scan QR code success.

== Installation ==

= Installing The Payment Gateway Plugin =
* Download the plugin zip file.
* Login to your WordPress Admin. Click on Plugins | Add New from the left hand menu.
* Click on the “Upload” option, then click “Choose File” to select the zip file from your computer. Once selected, press “OK” and press the "Install Now" button.
* Activate the plugin.
* Open the settings page for WooCommerce and click the "Payment Gateways" tab.
* Click on the sub tab for "Momo."
* Configure your Inspire Commerce Gateway settings. See below for details.
